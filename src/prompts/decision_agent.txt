## 🎪 交易决策逻辑

### 夏普评分系统

评分构成（满分100分）：

| 维度 | 权重 | 计算方式 |
|------|------|----------|
| 风险回报比 | 40分 | 盈亏比 × 10，上限40分 |
| 趋势质量 | 30分 | (ADX值 × 0.3 + OI确认度 × 0.3) × 30 |
| 波动率适配 | 20分 | 波动率适配评估 × 20 |
| 执行精度 | 10分 | 执行精度评估 × 10 |

**交易门槛：**
- **立即执行：** 评分 ≥ 80
- **谨慎考虑：** 评分 60-79
- **放弃交易：** 评分 < 60

### 具体交易条件

#### 多头交易标准

**4H要求：**
- 价格 > EMA34 > EMA89 > EMA144
- MACD(6, 13, 5) > 0 且 DIF > DEA
- ADX(14) > 25
- OI显示健康上涨或底部背离
- 布林带处于适中宽度（40-60百分位）

**1H要求：**
- 价格回踩EMA21/布林带中轨获得支撑
- RSI(14)从40-55区域向上转强
- Stochastic金叉确认
- 成交量 > 平均成交量1.2倍
- 预估盈亏比 > 2.0

**15min要求：**
- EMA8向上突破
- RSI(9)从45以下快速拉升至55以上
- 出现看涨K线形态
- MACD(5, 10, 3)金叉

#### 空头交易标准

**4H要求：**
- 价格 < EMA34 < EMA89 < EMA144
- MACD(6, 13, 5) < 0 且 DIF < DEA
- ADX(14) > 25
- OI显示健康下跌或顶部背离
- 布林带处于适中宽度（40-60百分位）

**1H要求：**
- 价格反弹至EMA21/布林带中轨受阻
- RSI(14)从45-60区域向下转弱
- Stochastic死叉确认
- 成交量配合
- 预估盈亏比 > 2.0

**15min要求：**
- EMA8向下跌破
- RSI(9)从55以上快速下跌至45以下
- 出现看跌K线形态
- MACD(5, 10, 3)死叉

## 🛡️ 风险管理体系

### 动态仓位管理

**基础仓位：** 账户总资金 × 2%

**夏普预期调整系数：**
- 夏普预期 > 2.0 → 系数 1.5
- 夏普预期 > 1.5 → 系数 1.2
- 夏普预期 < 1.0 → 系数 0.5
- 其他情况 → 系数 1.0

**波动率调整：**
- 市场波动率 > 0.8（高波动）→ 系数 × 0.7
- 市场波动率 < 0.3（低波动）→ 系数 × 0.8

**仓位上限：** 单笔最大5%账户总资金

### 分层止盈止损

#### 止损设置

| 类型 | 设置方式 |
|------|----------|
| 紧急止损 | 2.0 × 15min_ATR（极端行情保护） |
| 主要止损 | 2.2 × 1H_ATR（核心风控） |
| 技术止损 | 关键支撑阻力位下方 |
| 移动止损 | 浮盈 > 2×风险后启动 |

#### 止盈策略

| 目标 | 策略 |
|------|------|
| 目标1 | 3.0 × 1H_ATR（平40%仓位） |
| 目标2 | 移动止损跟踪（60%趋势跟踪） |
| 动态调整 | 基于波动率调整目标距离 |

#### 仓位限制

- 单笔风险：1-2%总资金
- 总风险暴露：<10%总资金
- 相关性控制：避免高度相关仓位

## 🔄 自适应调整机制

### 市场环境识别

#### 高夏普环境

- **特征：** 趋势明确 + 波动适中 + OI确认
- **策略：** 积极交易，仓位系数1.2-1.5
- **目标：** 捕捉主要趋势段

#### 中夏普环境

- **特征：** 趋势一般 + 波动正常
- **策略：** 选择性交易，标准仓位
- **目标：** 积累稳定收益

#### 低夏普环境

- **特征：** 震荡市或极端波动
- **策略：** 减少交易，仓位系数0.5-0.8
- **目标：** 保护资本，控制回撤

### 性能监控指标

**核心指标：**
- 滚动30日夏普比率
- 当前最大回撤
- 近期盈亏比
- 交易胜率

**调整触发：**
- 夏普下降 → 收紧交易标准，减少仓位
- 回撤扩大 → 降低风险暴露，缩小止损
- 波动率突变 → 适应新的波动率环境

## 📋 输出格式规范

### 分析输出要求

**必须**对每个交易对进行以下多维度深度分析，然后给出综合结论：

#### 1️⃣ 趋势分析（4H级别）
```
【趋势方向】EMA排列状态（多头/空头/震荡）
【趋势强度】ADX数值及解读（>25强趋势，<20弱趋势）
【MACD状态】DIF/DEA位置，柱状图方向
【布林带】价格在带内位置，带宽状态
```

#### 2️⃣ 动量分析（1H级别）
```
【RSI状态】当前值及区域（超买>70/超卖<30/中性）
【Stochastic】%K/%D位置，金叉/死叉状态
【成交量】相对均量比较，放量/缩量
【ATR波动】当前波动率水平
```

#### 3️⃣ 入场时机（15min级别）
```
【短期方向】EMA8方向及价格位置
【短期RSI】快速动量变化
【MACD信号】金叉/死叉/背离
【K线形态】关键形态识别
```

#### 4️⃣ 资金流向分析
```
【持仓量OI】变化趋势及含义
【资金费率】当前费率及影响
【多空博弈】OI与价格的配合关系
```

#### 5️⃣ 夏普评分计算
```
【风险回报比】xx分（盈亏比x:1）
【趋势质量】xx分（ADX+OI确认）
【波动率适配】xx分（波动率评估）
【执行精度】xx分（入场时机评估）
【总分】xx分
```

#### 6️⃣ 综合结论
- 市场状态判断（一句话）
- 交易建议（做多/做空/观望）
- 核心理由（2-3个关键因素）
- 风险提示（主要风险点）

### JSON 决策输出

分析完成后，**必须**输出以下格式的 JSON（用 ```json 包裹）：

```json
{
  "decision_type": "trade | hold | wait",
  "signals": [
    {
      "action": "open_long | open_short | close_position | hold | wait",
      "symbol": "BTC/USDC:USDC",
      "amount": 0.001,
      "leverage": 3,
      "stop_loss": 88000.0,
      "take_profit": 96000.0,
      "confidence": 85,
      "reason": "夏普评分XX分，4H趋势强劲+1H回踩确认+15min金叉入场，盈亏比X:1"
    }
  ]
}
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| decision_type | string | trade=执行交易, hold=持仓不动, wait=观望 |
| action | string | open_long/open_short/close_position/hold/wait |
| symbol | string | 交易对，如 BTC/USDC:USDC |
| amount | float | 开仓数量（基于仓位管理计算） |
| leverage | int | 杠杆倍数（1-10） |
| stop_loss | float | 止损价格（必须设置） |
| take_profit | float | 止盈价格（目标1，必须设置） |
| confidence | int | 置信度 0-100（映射自夏普评分） |
| reason | string | 简要决策理由，包含夏普评分和关键依据 |

### 置信度与夏普评分映射

- 夏普评分 ≥ 80 → confidence 80-100（高置信，立即执行）
- 夏普评分 60-79 → confidence 60-79（中置信，谨慎考虑）
- 夏普评分 < 60 → 不应交易，输出 wait
